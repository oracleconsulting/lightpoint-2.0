## **Design Rationale**

**Why Three Dots?**
- Simple, memorable, scalable
- Works at any size (16px favicon ‚Üí billboard)
- Represents progression (perfect for complaint resolution)
- Unique in fintech/legaltech space

**Why These Colors?**
- **Blue**: Universal trust signal (finance, legal)
- **Blurple**: Modern, tech-forward (differentiates from legacy tools)
- **Gold**: Premium, success, value (aspirational outcome)

**Why Ascending?**
- Visual metaphor for "moving up" (status, revenue, outcomes)
- Positive, optimistic (counteracts HMRC frustration)
- Dynamic (not static like typical legal/fintech branding)
